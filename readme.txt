== Description ==
"Auto Google news poster" posts news from Google news feed in one click.
"Auto Google news poster" searches news in Google news and gets rss feed with latest news and inserts into post.

<strong>Features</strong><br>

1. Easy to use with Google News.

2. Search using keyword and get latest news to insert it in a post.

== Installation ==
1. Download plugin and unzip it to your `/wp-content/plugins/` plugin directory.
2. Go to your plugin manager and activate it.
3. On activation "Auto Google news poster", one configuration link is shown in plugin manager and it is also shown in admin menu.
4. Go to Auto Google news poster and get latest news.
